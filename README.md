# IPX Sanitari Landing

File ini menyediakan layout landing page modern bertema sanitari.

Cara pakai:

- Buka landing/index.html di browser.
- Ubah palet warna di landing/styles.css pada variabel :root.
- Ganti teks, tautan, dan gambar placeholder sesuai kebutuhan.
- Struktur utama: header, hero, categories, enefits, eatured, pplications, process, cta, ooter.

Kustomisasi cepat:

- Logo: edit elemen .brand di index.html.
- Navigasi: ubah item di .nav-list.
- Kategori/Produk: duplikasi kartu pada masing-masing grid.
- Tipografi: ganti font di <link> Google Fonts di index.html.

Halaman tambahan:

- produk.html: katalog produk dengan filter kategori (chips).
- 	entang.html: profil perusahaan, nilai, sertifikasi, dan sejarah singkat.
